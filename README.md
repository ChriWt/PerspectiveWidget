# Requirements:
  - Python 3.13.x
  - CMake
  - C++ compiler (gcc)


# How to install
1) make a python virtual enviroment:
```
python -m venv .venv
```

2) Activate virtual enviroment:
  - Linux:
```
source .venv/bin/activate
```
  - Windows:
```
.\.venv\Scripts\Activate.ps1
```

3) Then compile cshape3d library:
```
pip install -e .
```

4) Install requirements:
```
pip install -r requirements.txt
```


# How to run
1) Activate virtual enviroment: (see above)

2) Run:
```
python application.py
```
